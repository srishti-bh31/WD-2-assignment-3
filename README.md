# Student Scoreboard (React App)

A modern React-based Student Scoreboard Application that allows users to manage student records, update scores dynamically, and view performance statistics with a clean cyberpunk-themed UI.

---

##  Overview

This project is built as part of a Web Development II (Advanced JS & React) lab assignment. It demonstrates the use of React functional components, hooks, props, and state management.

---

##  Features

- Display list of students with their scores  
- Add new students using a form  
- Update scores using input + save button  
- Pass/Fail status based on score  
- Live statistics dashboard:
  - Total Students  
  - Passed Students  
  - Average Score  
- Stylish cyberpunk UI
